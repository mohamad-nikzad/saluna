## Problem

Customers may prefer booking or interacting through Telegram or Bale instead of a web flow.

## Smallest Useful Version

Support one bot channel for browsing services and submitting an `AppointmentRequest`.

## Acceptance Criteria

- [ ] First bot channel is selected.
- [ ] Customer can find a salon or use a salon-specific entry point.
- [ ] Customer can submit an appointment request through the bot.
- [ ] Bot flow reuses existing public appointment request logic where possible.
