# Preset Catalog uses reusable library items

Catalog presets are assembled from reusable product-maintained preset categories and preset services, while each CatalogPreset keeps its own curated defaults for the assembled catalog. We chose this over editing one nested preset tree directly because platform staff need simple pages for maintaining reusable service catalog items, plus a separate assembly workflow for wiring those items into starter catalogs without forcing every preset to share the same prices, durations, or labels.
